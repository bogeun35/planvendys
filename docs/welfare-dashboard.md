# 복지대장 지급현황 대시보드

## 개요

복지대장 서비스의 고객사별 복지 포인트 지급 현황을 조회하는 대시보드입니다.
Redash 쿼리 기반으로 실시간 데이터를 제공합니다.

## 사용 방법

### 기간 조회
- **시작일/종료일**을 선택하고 조회 버튼을 클릭합니다
- 핫키 버튼으로 빠르게 기간을 설정할 수 있습니다: 오늘, 이번달, 지난달, 올해 등

### 서머리 카드
상단에 표시되는 요약 정보입니다:
- 총 지급액
- 고객사 수
- 사용자 수
- 건수
- 인당 평균

### 인사이트 패널
- **일자별 추이**: 날짜별 지급 현황을 막대 차트로 표시
- **상위 고객사 집중도**: 파레토 분석으로 상위 N개사의 점유율 확인
- **규모 분포**: 금액 구간별 고객사 수 분포

### 고객사 테이블
- 검색: 고객사명으로 검색 가능
- 정렬: 헤더 클릭으로 정렬 변경
- 복사: 클립보드 버튼으로 엑셀 호환 데이터 복사

### 상세 정보
테이블에서 고객사를 클릭하면 우측에 상세 정보가 표시됩니다:
- 날짜별 지급 요약
- 사용자별 지급 내역

## 전년 비교
전년 동기간 데이터를 자동으로 조회하여 비교 분석합니다:
- 지급액/사용자 증감률
- 신규 진입(`NEW`) / 이탈(`OUT`) 고객사

## 데이터 내보내기

| 기능 | 방법 |
|------|------|
| 전체 복사 | 클립보드 버튼 클릭 |
| 범위 복사 | 셀 드래그 선택 후 Ctrl+C |

복사된 데이터는 탭 구분 텍스트로 엑셀에 바로 붙여넣기 가능합니다.
