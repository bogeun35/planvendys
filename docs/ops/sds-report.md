# 식권대장 결제내역 보고서 가이드

## 개요

특정 고객사(comId)의 식권대장 결제내역을 부서별로 조회하고 엑셀로 내보내는 도구입니다.
SDS 전용 식대상세내역 보고서 생성에 사용됩니다. Redash 쿼리(QID: 1353) 기반입니다.

## 사용 방법

### 1. 조회 조건 설정
- **comId**: 고객사 UUID 입력 (기본값: SDS comId)
- **시작일/종료일**: 조회 기간 설정
- 핫키: 지난달, 지지난달, 이번달

### 2. 조회 실행
- "조회" 버튼 클릭
- 7일 단위 청크로 분할 조회 (대용량 데이터 안정성 확보)
- 진행률 바와 구간별 건수가 실시간 표시
- 최대 조회 기간: 185일
- "중단" 버튼으로 진행 중 조회 취소 가능

### 3. 서머리 카드
조회 완료 시 상단에 요약 정보 표시:
- 총 건수, 사용자 수
- 부서1 수, 부서2 수
- 선택 정책 사용식대, 총 결제총액, 총 정산금액

### 4. 2단 레이아웃

**좌측: 부서2 리스트**
- 부서2 기준으로 그룹핑된 목록
- 각 부서의 건수 표시
- 클릭하여 해당 부서 데이터만 필터링
- "DL" 버튼으로 개별 부서 엑셀 다운로드

**우측: 결제 내역 테이블**
- 선택된 부서(또는 전체)의 상세 결제 내역
- 5,000건 초과 시 자동 잘림 (엑셀 다운로드로 전체 확인)

## 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| 결제번호 | 결제 고유번호 |
| 날짜 | 결제 일시 |
| ID | 사용자 ID |
| 이름 | 사용자명 |
| 사원번호 | 사원번호 |
| 부서1~3 | 부서 계층 |
| 가맹점 | 결제 가맹점명 |
| 선택 정책 사용식대 | 복지 포인트 사용액 |
| 총 결제총액 | 결제 총액 |
| 정산금액 | 정산 대상 금액 |
| 결제형태 | 결제 방식 |
| 조식/중식/석식/특식 | 식사 구분별 금액 |

## 데이터 내보내기

- **전체 엑셀 다운로드**: "전체" 시트 + 부서2별 개별 시트 포함
- **부서별 다운로드**: 좌측 리스트의 "DL" 버튼으로 개별 부서 엑셀
- **클립보드 복사**: 현재 보이는 테이블 데이터를 탭 구분 텍스트로 복사

## 주의사항

- 7일 단위 분할 조회로 Redash 타임아웃을 방지합니다
- 폴링 대기 시간이 길어질 수 있으며, 최대 120회(약 3분)까지 대기합니다
- 실패한 구간은 로그에 빨간색으로 표시되며, 해당 구간 데이터는 누락됩니다
- comId를 변경하여 다른 고객사 조회도 가능합니다
