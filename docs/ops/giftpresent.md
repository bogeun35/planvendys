# 단체선물대장 주문현황 가이드

## 개요

단체선물대장 서비스의 주문 현황을 다각도로 분석하는 대시보드입니다.
Redash 쿼리(QID: 1335) 기반으로 전체/고객사별/상품별/매입처별/일자별 현황을 제공합니다.

## 사용 방법

### 기간 조회
- 시작일/종료일 선택 후 "조회" 클릭
- 핫키: 이번달, 지난달, 지지난달, 올해, 작년
- 기본값: 2026-01-01 ~ 2026-02-28

### 탭 구성

#### 전체 현황
- 개별 주문 건 단위 목록
- 상태별 필터링 (미선택/선택완료/주문확정/배송완료)
- 고객사, 상품명, 제휴점, 수령인 통합 검색
- 미선택/선택완료 건은 경고색(붉은 배경)으로 표시
- 100건 단위 페이지네이션

#### 고객사별
- 고객사 단위 집계 (건수, 상태별, 금액, 마진)
- 선택률과 진행 상태 바 시각화
- 행 클릭 시 상세 패널: 상품별 분포 + 미선택 목록

#### 상품별
- 상품코드 단위 집계
- 고객사 수, 판매/매입/마진 분석
- 행 클릭 시 상세 패널: 고객사별 분포

#### 매입처(제휴점)별
- 제휴점 단위 집계
- 상품 종류, 평균 단가 포함
- 행 클릭 시 상세 패널: 상품별 + 고객사별 분포

#### 일자별 현황
- 확정일자 기준 일별/주별/월별 그룹핑
- 차트 2종: 확정 건수 추이(막대+누적 라인), 확정 금액 추이(스택 막대+누적 라인)
- 기간 필터로 특정 구간만 조회 가능
- 미확정(일자없음) 건은 별도 행으로 표시

## 금액 계산 규칙

| 항목 | 계산 방식 |
|------|----------|
| 판매금액 | 미선택 제외, totalPrice 합산 |
| 매입금액 | 미선택 제외, supplyPrice 합산 |
| 마진 | 판매금액 - 매입금액 |
| 마진율 | (마진 / 판매금액) x 100 |
| 예상금액 | 미선택 건의 memo 필드 숫자 파싱 |

## 데이터 내보내기

- **Excel 다운로드**: 전체/고객사별/상품별/제휴점별 시트 포함
- **클립보드 복사**: 각 탭의 집계 테이블을 탭 구분 텍스트로 복사

## 상세 패널

고객사별/상품별/매입처별 탭에서 행 클릭 시 우측 슬라이드 패널이 열립니다.
- 요약 카드 (건수, 선택률, 금액)
- 하위 분포 테이블
- 미선택 목록 (고객사 상세에서만)
- Esc 키 또는 오버레이 클릭으로 닫기
