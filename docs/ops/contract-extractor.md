# 수기 계약서 추출기 가이드

## 개요

식권대장 제휴점 서비스공급 계약서 PDF에서 주요 데이터를 자동 추출하여 Excel로 변환하는 도구입니다.

## 사용 방법

### 1. PDF 업로드
- 드래그 앤 드롭 또는 클릭하여 PDF 파일 선택
- 여러 파일 동시 선택 가능
- 중복 파일은 자동 필터링

### 2. 전체 추출
- "전체 추출" 버튼 클릭 시 모든 PDF를 순차 처리
- 각 파일에서 계약서 마지막 페이지의 정보를 파싱
- 추출 진행률이 상단 프로그레스 바에 표시

### 3. 추출 항목

| 항목 | 설명 |
|------|------|
| 계약체결일 | YYYY-MM-DD 형식 |
| 상호 | 제휴점 상호명 |
| 대표자 | 대표자명 |
| 매장전화 | 매장 전화번호 |
| 휴대전화 | 대표자 휴대전화 |
| 주소 | 제휴점 주소 |
| 이메일 | 이메일 주소 |
| 운영시간 | 매장 운영시간 |
| 은행명 | 정산 은행 |
| 계좌번호 | 정산 계좌 |
| 명의주 | 계좌 명의 |
| 사업자번호 | 사업자등록번호 |

### 4. 네이버 Place 연동
- PDF 추출 완료 후 자동으로 네이버 Place 검색 실행
- 주소와 상호명을 기반으로 유사도 매칭
- 일치율: 60% 이상 녹색, 30~60% 노란색, 30% 미만 빨간색
- 테이블의 Place 열에서 일치율 확인 가능

### 5. 상세 패널
- 테이블에서 행 클릭 시 우측에 상세 정보 표시
- Place 검색 결과와 네이버 플레이스 링크 제공
- 바로가기 버튼: 제휴점/구정산정보/거래처/계약서 만들기

## 데이터 내보내기

- **엑셀 다운로드**: 전체 추출 데이터 + Place URL + 일치율 포함
- **클립보드 복사**: 탭 구분 텍스트, 엑셀에 바로 붙여넣기 가능

## 로그 / 디버그

- 하단 로그 패널에서 처리 과정 확인
- "Raw" 버튼으로 PDF 원본 텍스트 좌표 데이터 확인 가능 (파싱 오류 디버깅용)

## 주의사항

- 계약서 양식이 표준 형식과 다를 경우 일부 항목이 누락될 수 있습니다
- 누락 항목은 로그에 표시되며, 엑셀에서 수동 보정이 필요합니다
- PDF.js와 SheetJS 라이브러리를 CDN에서 로드합니다
